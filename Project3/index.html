Our Print(finger) Project

For this project we decided to research the role that machine learning algorithms play in fingerprint recognition since it is a commonly used identification or security sure. Fingerprint recognition is used every day for user-authentication, employee background checks, secure facility access, and many more applications. We wanted to see if we could find new use for the fingerprint images and explored predictions that we could make based on the images provided in the database.

We utilized the Sokoto Coventry Fingerprint Dataset (SOCOFing), which is a biometric fingerprint database made up of 600 participants. The dataset also contained the following attributes: hand, finger, gender. We utilized a neural network to see to attempt to predict characteristics of a person (i.e. hand, gender, finger) by looking at an image of their fingerprint.
